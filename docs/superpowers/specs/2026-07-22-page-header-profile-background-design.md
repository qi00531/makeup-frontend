# 页面英文标题与个人页背景设计

## 目标

- 首页顶部增加英文标题 `MY BEAUTY STUDIO`，视觉规则与知识库的 `MY BEAUTY ARCHIVE` 保持一致。
- “我的”页面顶部删除中文“我的”，仅保留 `MY BEAUTY PROFILE`。
- “我的”页面使用项目根目录 `图片资产/知识库页 (1).png` 作为背景来源，并采用与知识库相同的渐变遮罩透明度。

## 实现方案

将选定图片复制到 `src/assets/bg-profile-flower.png`，避免运行时引用源码目录之外的文件。首页和个人页仅调整页面标题及背景样式，不改变卡片结构、交互逻辑、底部导航或后端服务接口。

## 验收标准

1. 首页可见且仅有一个 `MY BEAUTY STUDIO` 页面标题。
2. 个人页可见 `MY BEAUTY PROFILE`，页面顶部不再显示中文一级标题“我的”。
3. 个人页背景引用项目内复制的图片资产，并使用与 `.library-page` 相同的渐变遮罩参数。
4. 现有页面流程、导航、响应式尺寸、测试、构建和 lint 均保持正常。
