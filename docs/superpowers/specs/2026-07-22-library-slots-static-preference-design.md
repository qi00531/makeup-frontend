# 知识库卡位与静态偏好风格设计

## 目标

精简知识库资产内容和“我的”页面偏好风格展示，同时保持现有页面布局、筛选标签、导航与预制教程跳转逻辑。

## 知识库资产标题

- 教程页签内容区不再显示“教程资产”标题和数量。
- 部位页签内容区不再显示“部位资产”标题和数量。
- 页签、筛选标签、筛选按钮和资产卡片布局保持不变。
- 空结果仍显示现有空状态。

## 部位素材卡位

- 部位筛选标签保持原有内容：全部、底妆、眼妆、腮红、修容、唇妆。
- 部位素材数据只保留三个预制卡位：眼妆一个、修容一个、唇妆一个。
- “全部”筛选显示三个卡位。
- “眼妆”“修容”“唇妆”分别只显示对应的一个卡位。
- “底妆”“腮红”筛选显示空状态。
- 三个卡位继续只显示预制视频封面，点击进入各自对应的预制图示教程。
- 混搭页需要的底妆、腮红等预制选项不应因知识库卡位精简而丢失；知识库展示素材与混搭选择素材在数据层明确区分。

## 偏好风格

- “我的”页面中的“偏好风格”改为静态信息行。
- 静态行保留图标、标题和当前偏好风格文字。
- 删除点击行为、展开状态、选项列表和右侧箭头。
- 当前风格仍从本地个人资料读取，但此页面不再提供修改入口。
- 学习提醒和隐私与数据保持现有交互。

## 验收

- 页面中不出现“教程资产”或“部位资产”。
- 部位“全部”只有三个卡片，眼妆、修容、唇妆各一个。
- 原有六个部位筛选标签仍全部存在。
- 底妆与腮红筛选显示空状态。
- 偏好风格不再具有按钮角色，不能展开，页面中不存在偏好风格选项列表。
- 现有混搭选择、预制加载、效果预览和教程流程不受影响。
