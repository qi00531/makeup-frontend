# 美妆教程扩展页面前端设计

## 1. 范围

在现有上传、照片确认、解析、适配预览四页基础上，新增以下页面：

1. 微调输入页
2. 图示教程页
3. 眼部精讲页
4. 知识库页
5. 混搭编辑页

明确不开发跟练页。删除底部导航中的“跟练”，导航改为：首页、知识库、混搭、我的。“我的”因参考图没有提供页面结构，继续保留轻量占位页。

页面结构以用户提供的十屏参考图为最高优先级，产品文档只补充图片未表达的交互和数据边界。

## 2. 路由与流程

- `/adjust`：微调输入页。
- `/tutorial`：图示教程页。
- `/eyes`：眼部精讲页。
- `/library`：知识库页。
- `/mix`：混搭编辑页。
- `/profile`：我的占位页。
- 移除 `/practice` 页面和对应导航入口。

流程：

```text
适配预览
├─ 适合我 → 图示教程
└─ 需要微调 / 不适合我 → 微调输入 → 图示教程
                                      └─ 眼部精讲

知识库 → 选择部位资产 → 混搭编辑 → 生成图示教程
```

## 3. 微调输入页

结构遵循参考图：

1. 顶部返回按钮和“微调设置”标题。
2. 个人风格选择。
3. 适配场合选择。
4. 执行工具多选。
5. 自由文本补充。
6. 主操作“生成方案”。
7. 四项底部导航。

个人风格和适配场合为单选，执行工具为多选，自由文本可为空。点击生成方案后保存选择结果并进入图示教程。

## 4. 图示教程页

结构遵循参考图：

1. 顶部返回按钮和“图示教程”标题。
2. 脸部底图及妆容图层。
3. 右侧图层色卡。
4. 步骤进度时间轴。
5. 步骤列表。
6. 产品信息与视频切片入口。
7. 新手/熟练模式切换。

点击步骤后更新当前步骤、脸部高亮图层和产品说明。模式切换只改变说明密度，不改变教程数据。眼妆步骤提供“查看眼部精讲”入口。

## 5. 眼部精讲页

结构遵循参考图：

1. 顶部返回按钮和“眼部精讲”标题。
2. 放大眼部底图。
3. 所有重点区域集中叠加在同一底图。
4. 可点击区域标签。
5. 视频切片进度条。
6. 上一步、下一步操作。
7. 当前区域说明和适配建议。
8. 四项底部导航。

区域包括眼影、眼线、卧蚕、睫毛、眼头提亮、下至和眉眼距离。点击区域后只改变高亮与说明，不跳转到独立页面。

## 6. 知识库页

结构遵循参考图：

1. 顶部标题与搜索框。
2. 教程、部位、产品三类视图。
3. 风格、场景、难度等筛选入口。
4. 九宫格资产卡片。
5. 四项底部导航并高亮知识库。

搜索和分类切换在前端真实过滤数据。点击部位资产可进入混搭页，并将资产作为预选项带入。

## 7. 混搭编辑页

结构遵循参考图：

1. 顶部返回按钮和“混搭编辑”标题。
2. 中央脸部底图。
3. 围绕脸部的部位槽位。
4. 已选择资产列表。
5. 兼容性提示。
6. 主操作“保存方案”或“生成教程”。
7. 四项底部导航并高亮混搭。

部位槽位包括底妆、眉毛、眼妆、腮红、修容、高光和唇妆。用户可以切换预置资产；若选择存在风格冲突，页面显示具体兼容性建议。生成后进入图示教程。

## 8. 数据与后端边界

新增共享类型：

- `AdjustmentRequest`：个人风格、场合、工具和自由文本。
- `TutorialStep`：步骤、部位、产品、图层颜色、视频切片和说明。
- `IllustratedTutorial`：标题、模式、步骤和当前进度。
- `EyeRegionGuide`：眼部区域、图层位置、切片和适配建议。
- `LibraryAsset`：资产 ID、分类、来源、风格、场景、难度和缩略图。
- `MixSelection`：各部位选择的资产 ID。
- `CompatibilityHint`：冲突类型、说明和调整建议。

本期继续提供前端本地数据服务。页面通过服务接口获取数据，不在组件内硬编码未来 API 地址。后端接入时替换服务实现，保留页面结构。

## 9. 视觉与交互

- 延续现有粉色矿物、苔藓绿、纸张拼贴和低饱和暖红视觉体系。
- 新增页面不得改变现有四页结构。
- 使用同一套 Lucide 图标、卡片圆角和语义颜色。
- 关键触控区域不小于 44px。
- 320px 至 430px 宽度无横向滚动。
- 仅使用有意义的步骤切换和高亮动画，并支持减少动态效果。

## 10. 验收标准

- 微调页所有选择项和自由文本可操作，生成方案可进入图示教程。
- 图示教程可切换步骤、模式、图层说明并进入眼部精讲。
- 眼部精讲的七个区域可切换，高亮与说明同步更新。
- 知识库搜索、分类与筛选能够真实过滤资产。
- 知识库资产可预选到混搭页面。
- 混搭部位可切换资产，并显示兼容性建议。
- 混搭方案可生成图示教程。
- 底部导航为首页、知识库、混搭、我的，没有跟练入口。
- `/practice` 路由被移除。
- 新增页面具有对应组件测试和完整流程测试。
