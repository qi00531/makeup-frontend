# 妆容微调问卷重构设计

## 目标

保持“需要微调”页面现有视觉样式、选中反馈、生成按钮和教程跳转逻辑，删除旧问题及自由文本，改为三个无编号部分。

## 问卷结构

- 个人风格：风格（多选）、使用场景（多选）。
- 脸部匹配：保留教程部位（多选）、肤质（单选）、希望修饰的问题（多选）。
- 工具：限制条件（多选）。

## 数据契约

`AdjustmentRequest` 调整为 `styles`、`occasions`、`retainedParts`、`skinType`、`concerns`、`constraints`。所有多选字段使用字符串数组，肤质使用字符串。页面允许用户按现有逻辑直接生成方案；肤质默认选择“混合性肌肤”，其余多选项默认为空。

## 兼容范围

同步更新本地学习服务、测试与后端对接说明。路由仍为 `/preview → /adjust → /tutorial`，不调整页面结构、导航或教程生成交互。
